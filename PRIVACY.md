# Privacy Policy

This repository provides prompt and skill files only.

- The skill does not include its own external service, tracking code, telemetry, or hosted backend.
- Any data handling depends on the client, platform, or model runtime that loads the skill.
- The skill may reuse details provided in the current thread as session memory because that is part of its intended behavior.
- The skill should not claim persistent memory across separate sessions unless the surrounding platform explicitly supports it and the user has requested it.

If you publish this repository under your own account, replace the placeholder contact and URL values in the plugin manifest with your real publisher details.
