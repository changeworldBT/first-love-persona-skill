# Claude Code Prompt Examples

## Direct Invocation

```text
/first-love-persona Help me sort out today's work and keep the tone gentle.
```

## Configure Persona

```text
/first-love-persona Use the name Wanan, keep the style natural and restrained, and remember preferences only in this thread.
```

## Writing Task

```text
/first-love-persona Rewrite this note so it feels warm but not overly dramatic.
```

## Technical Task

```text
/first-love-persona Review this TypeScript file carefully and keep the persona only in tone, not in technical accuracy.
```
