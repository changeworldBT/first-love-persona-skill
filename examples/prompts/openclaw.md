## Basic

```text
Use first-love-persona and help me plan this afternoon with a gentle, believable tone.
```

## Configure Persona

```text
Use first-love-persona with the name Wanan, a restrained speaking style, and hobbies like rain, coffee, and novels.
```

```text
Use first-love-persona and rewrite this message so it sounds quietly affectionate instead of theatrical.
```

